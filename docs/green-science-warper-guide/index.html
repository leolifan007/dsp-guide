You want that.</p> <div class="callout"> <strong>TL;DR</strong> — Green Matrix needs 3 things:<br> <strong>Processors Casimir Crystals EM Turbines.</strong><br> Scale your existing Processor line. Each vessel trip costs <strong>1 warper</strong>. One warper moves <strong>200 items</strong>. That&rsquo;s insanely efficient.</p> <p><strong>Build a warper production line right next to your green matrix setup.</strong> Route a fraction of your green matrix output into Assembling Machines M2/M3 that produce warpers. Feed them into an ILS. Set ILS slot to Supply.</p> <p>Now every ILS in your network can grab warpers and start shipping across the galaxy.</p> <div class="pro-tip"> <strong> One gotcha:</strong> ILS ships won't pick up warpers unless you set a slot to Supply for warpers.
